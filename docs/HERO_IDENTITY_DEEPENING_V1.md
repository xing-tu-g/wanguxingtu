# Hero Identity Deepening Sprint v1

日期：2026-07-05

## Scope

本阶段只做英雄玩法深度强化，不新增系统，不改战斗规则，不改星力结构，不改职业体系，不改匹配系统。

新增隐藏标签配置：

- `data/hero_identity_deepening.json`

该配置只描述英雄节奏与身份，不参与 Battle 规则结算。

## Hidden Tag Definition

| tag | 含义 | 典型体验 |
| --- | --- | --- |
| `burst` | 爆发型 | 快速制造击杀、破口或关键伤害。 |
| `sustain` | 持续型 | 通过燃烧、治疗、护盾或续航拖出优势。 |
| `control` | 控制型 | 通过眩晕、减速、削攻影响敌方节奏。 |
| `growth` | 成长型 | 越拖越强，鼓励保护和拖回合。 |
| `counter` | 反击型 | 承压、防守、护卫后反打。 |
| `operation` | 运营型 | 通过机动、指令、召唤、支援改变战场资源。 |

## Low Presence Watchlist

这些英雄不是判定失败，而是需要重点验证“玩家是否能感知到技能价值”：

- 赵云：自动部署低估高移动切入价值；先强化突进命中反馈，必要时追伤 `1 -> 2`。
- 张飞：护卫价值容易被玩家忽略；先强化减伤归属提示，必要时扩大护卫判断范围。
- 马超：与赵云同为快攻近战，需要突出真伤破甲；先强化破甲提示，必要时提高首击收益。
- 华雄：贴身 `front_1` 容易错过触发；先强化斩将反馈，必要时改为前方同排。
- 徐盛：长线护盾不醒目；先强化持续护盾 UI，必要时部署盾 `2 -> 3`。

## Same-Class Differentiation

- 战士：赵云/马超/孙策/颜良偏爆发，关羽/魏延/孙坚/夏侯惇偏成长，张辽/凌统/袁绍偏运营，马岱/文丑偏控制。
- 法师：周瑜/陆逊/贾诩偏持续灼伤，司马懿/郭嘉/甄姬/貂蝉偏控制，曹操/诸葛亮/鲁肃/小乔偏运营。
- 坦克：张飞/许褚/典韦/黄盖偏反击或承压，曹仁/周泰/高顺/董卓/徐盛偏持续守线。
- 射手：黄忠偏超远爆发，孙尚香偏持续点燃，太史慈偏真伤破防，公孙瓒偏机动号令。
- 刺客：乐进偏高风险背刺，甘宁偏稳定奇袭爆发。

## Suggested Micro-Tuning Only

本阶段不直接修改数值。允许进入后续微调池的只有：

- 固定伤害 `+1` 或 `-1`。
- 护盾/治疗固定值 `+1` 或 `-1`。
- 持续回合 `+1` 或 `-1`。
- 触发条件从过窄改为同类已有目标范围。

禁止：

- 百分比。
- 新资源。
- 新职业。
- 新战斗规则。
- 新 AttackShape 结构。
- 新系统性机制。
