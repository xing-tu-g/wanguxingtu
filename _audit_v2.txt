第一章：项目概述 1.1 核心乐趣假设 在华夏文明的星图上，用三国名将的魂棋进行一场自动战棋对决——选将、布阵、看他们在随机地形上自动厮杀，赢了就是你把一颗陨落的将星拉回了文明。 1.2 设计支柱 1. 星图设定——所有玩法围绕&quot;星图&quot;&quot;魂星&quot;&quot;天缺&quot;展开，不做换皮三国 2. 易学不易精——8-15分钟上手，但阵容搭配和地形利用需要长期钻研 3. 每局新棋局——随机地形和随机策略卡让每局截然不同，拒绝背板 4. 降肝降氪——每日首胜加前十局收益最大，后续递减，不靠在线时长绑架玩家 5. 真实概率——抽卡概率透明公示，保底机制清晰 1.3 产品定位 休闲 + 轻策略 + 爽感 • 休闲：8-15 分钟一局，异步回合制 • 轻策略：武将搭配、地形利用、策略卡选择 • 爽感：自动战斗演出、技能爆发、弈天师生命见底的胜利反馈、每个页面的Q版美术 1.4 核心循环 获得武将 → 编辑卡组 → 对战 → 结算领奖。武将升级 ← 抽卡 ← 资源获取。天梯排位。 1.5 反痛点设计 • 降肝降氪 • 控制武将升级增加的数值与技能提示效果 • 每日首胜加前十局收益最大，后续递减 1.6 凭什么可能火 1. 地形随机刷新——每局地图不同，拒绝背板 2. 每个武将设计有特色——技能差异化，无凑数卡 3. 抽卡纯随机——真实概率公示 4. 收益递减保护——轻肝友好，不劝退休闲玩家 第二章：世界观 2.1 万古星图 在时间诞生之前，天地间悬浮着一幅名为「万古星图」的棋盘。它不记年月，不辨朝代。大禹治水的凿痕、韩信暗度陈仓的马蹄、李白醉后的孤篇、哪吒闹海的混天绫——所有曾在华夏文明中绽放过光芒的存在，都在星图上化作一枚魂星。千万年来，四百余颗魂星彼此辉映，织成文明的经纬。 2.2 天缺 建兴十二年秋，五丈原。诸葛亮禳星那夜，七星灯不是被魏延踢翻的——是裂痕。一道看不见的裂痕撕裂了星图，七星灯应声而碎。不止诸葛亮的星落了。短短数十年间，关羽、张飞、周瑜、典韦、郭嘉——那些本不该早逝的人——他们的魂星没有像往常那样回到星图循环，而是被裂痕吞噬，散落在时空的裂缝中。这便是「天缺」。 2.3 弈天师 裂痕出现的同一年，世间开始有人觉醒一种能力：感知星图中的魂星，将其凝结为「命魂棋」，在虚空中落子为战。这些人被称作弈天师。继承诸葛亮未竟事业的弈天师，从三国开始觉醒。每获得一枚命魂棋，就是将一位传奇从遗忘的边缘拉回；每在棋盘上赢得一场战斗，星图上的裂痕就愈合一线。 2.4 三国是起点 裂痕虽然贯穿万古——它吞噬的远远不止三国。春秋的魂星、楚汉的将魂、神话时代的神格，全在裂缝中。但没有人能感知到三国之前和之后的缺失。因为裂痕抹去的不是记忆，而是&quot;它们曾经存在&quot;的事实本身。只有三国。因为裂痕正是在这一代人眼前撕开的。这一代人亲眼看见关羽之星陨落、看见诸葛亮禳星失败、看见三国归晋时四百将星一夜俱寂。所以弈天师最初只能感知三国时代的魂星——那是裂痕的起点，是星图最脆弱的伤口，也是唯一还能被观测到的地方。 第三章：对战系统 3.1 通用规则 • 横屏需适配安卓各类手机屏幕 • 棋盘：10 列 × 5 行（玩家部署区 1–3 列，敌方部署区 8–10 列，中场 4–7 列） • 双方各一名弈天师，初始 30 生命值 • 弈天师为纯血量目标，所有职业对其伤害一致 • 先将对方弈天师生命值打至 0 者胜 3.2 回合流程 1. 先后手抽取——先手初始 5 星力，后手初始 6 星力 2. 先手方回合：放置武将（消耗星力）→ 武将自动移动 → 武将自动攻击 → 回合结束 3. 后手方回合：放置武将（消耗星力）→ 武将自动移动 → 武将自动攻击 → 回合结束 4. 循环，直到一方弈天师生命值归零 • 星力是弈天师召唤武将与发动战术的战场能量 • 基础每回合恢复 2 点星力（上限 10） • 每 3 个回合进入一次「星潮」：后续每回合星力恢复量加 1，并额外抽取 1 张策略卡 • 星潮会强化压迫：第 8/12/16... 回合起，单位对弈天师伤害分别 +1/+2/+3...，只影响攻击弈天师，不影响武将对打 • 无回合上限，打到弈天师生命归零为止 • 三阶段推进：己方半场 → 中场 → 敌方弈天师 • 弈天师无固有减伤，所有单位攻击力全量作用于弈天师 3.3 部署规则 • 玩家可在己方半场任意空格部署武将 • 星力等于武将召唤消耗值 3.4 战斗内查看 •
