# Battle FX Runtime

Runtime effects are layered by Godot code and kept separate from hero art.

Planned folders:

- `hit/` for impact bursts and hit-stop accents.
- `slash/` for melee arcs.
- `dash/` for movement streaks.
- `fire/` for burn and flame skills.
- `lightning/` for thunder/control skills.
- `heal/` for recovery effects.
- `buff/` for shields and stat boosts.

Current Battle Polish v1 uses procedural placeholders in `BattleAnimator.gd`.
